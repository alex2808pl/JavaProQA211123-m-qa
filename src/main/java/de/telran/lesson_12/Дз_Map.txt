2*) У вас есть приют для животных, в которых живут коты.
Есть объект Cat с данными: имя, возраст, вес, порода.
- создайте Map, ключем которого будет имя, а значением объект Cat.
- информация о котах должна храниться в упорядоченном виде по имени.
- выведите имена котов, порода которых равна N (на ваше усмотрение).
- выведите список всех котов, вес которых больше значения М (на ваше усмотрение).
- * найдите кота с самым большим весом и возрастом одновременно.

3**) Найти маршрут из заданного списка билетов, желательно используя Map.
Учитывая список билетов, найти маршрут по порядку, используя данный список.
Вход:
«Berlin» -> «London»
«Tokyo» -> «Seoul»
«Mumbai» -> «Berlin»
«Seoul» -> «Mumbai»
Выход:
Tokyo->Seoul, Seoul->Mumbai, Mumbai->Berlin, Berlin->London

4**) Сгруппируйте слова с одинаковым набором символов, желательно используя Map.
Дан список слов со строчными буквами. Реализуйте функцию поиска всех слов с одинаковым уникальным набором символов. 
вход: String words[] = {"student", "students", "dog", "god", "cat", "act", "flow", "wolf"};
Выход: 
student, students, 
cat, act, 
dog, god, 
flow, wolf
